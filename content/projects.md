A collection of the custom scripts, security tools, and hardware experiments I've built in my free time.

---

### <a href="https://github.com/br0wnboi/ned-cvss-calculator" target="_blank">ned-cvss-calculator</a>
Ned is a privacy-focused, locally-run browser extension for calculating CVSS v3.1 and v4.0 scores directly in your browser without ever making an API call.

### <a href="https://github.com/br0wnboi/hugo-theme-symbiote" target="_blank">hugo-theme-symbiote</a>
The very theme powering this website! A dark, minimalistic, highly-optimized, terminal-inspired Hugo theme built for security researchers and hackers. 

### <a href="https://github.com/br0wnboi/thmate" target="_blank">thmate</a>
Your terminal-based TryHackMe-Mate. A CLI tool to instantly find out your THM rank, completed rooms, and other profile details right from your terminal.

### <a href="https://github.com/br0wnboi/rpi-wifi-tester" target="_blank">rpi-wifi-tester</a>
A custom Raspberry Pi build functioning similarly to a WiFi Pineapple. Please don't judge the code quality too harshly—it was my college major project, and I'm legally obligated to leave it up to prove I graduated.
